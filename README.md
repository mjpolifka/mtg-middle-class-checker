# MTG Rare Printing Finder (Web App)

This project now includes a Flask web app version of the deck analyzer.

## Run locally

1. Install dependencies:
   - `pipenv install`
2. Start the app:
   - `pipenv run python app.py`
3. Open:
   - `http://127.0.0.1:5000`

## Usage

1. Enter an Archidekt deck ID.
2. Submit.
3. Review cards in the deck that have rare/mythic printings (excluding sets in the ignored list).
